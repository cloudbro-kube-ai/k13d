주요 기능

  • NEW REPL 구현 (1,554줄)
  • NEW 한글(UTF-8) 입력 지원
  • NEW Left/Right 커서 네비게이션
  • NEW Ctrl+Enter 개행 입력
  • NEW 도구 승인 프롬프트 (Y/A/n/q)
  • NEW K8s 컨텍스트 인식 AI
  • NEW Tab 자동 완성
  • NEW 명령 히스토리 (Up/Down)
# CLI 실행
$ ./k13d --cli

# AI 명령어
> :ai 파드 상태 확인

# 리소스별 AI 분석
> :ai pod/nginx why failing?

# kubectl 패스스루
> get pods -n production

:model [<이름>]LLM 모델 조회/전환
:namespace [<이름>]네임스페이스 조회/설정
:context [<이름>]K8s 컨텍스트 조회/전환
:mcp [list|tools]MCP 상태/도구 목록
:history명령 히스토리 표시
