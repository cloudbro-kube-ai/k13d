# AI 입력 영역 전체화면 확대 방지

## 문제
자연어 입력창(AI Panel)을 전체화면으로 볼 때 입력폼 영역(`ai-input-container`)이 `height: 30vh`로 설정되어 화면의 30%를 차지함.

## 원인
`css/views.css:1234` 및 `bundle.css:5104`에서 `.ai-panel.expanded .ai-input-container`에 `height: 30vh` 적용 중.

## 수정 대상 파일
1. `pkg/web/static/css/views.css` (라인 1233-1241)
2. `pkg/web/static/bundle.css` (라인 5103-5111)

## 변경 내용
- `height: 30vh` 제거 → 원래 높이(콘텐츠 기반) 유지
- `padding: 30px 40px` → `padding: 18px 40px` (가로 패딩만 확장, 세로는 기존 유지)

## 검증
- `make build` 후 Web UI 실행
- AI 패널 열기 → 전체화면 버튼 클릭
- 입력 영역이 기존 높이를 유지하는지 확인
